Apa itu Tipe Data?

üìö Definisi

Tipe Data (Data Types) adalah classification yang menentukan jenis nilai yang bisa disimpan dalam variable dan operasi apa yang bisa dilakukan pada nilai tersebut. Think of it as "aturan main" untuk setiap variable - apakah itu number, text, true/false, atau collection of items.

‚úÖ Mengapa Tipe Data Penting?

1. Type Safety

Mencegah operasi invalid: "5" + 3 adalah error karena string + int tidak kompatibel.

2. Memory Efficiency

Different types use different memory. Integer lebih efficient dari string untuk numbers.

3. Code Clarity

Type hints make code self-documenting: mass: float jelas lebih informative.

4. Performance

Right type = faster operations. NumPy arrays jauh lebih cepat untuk math operations vs lists.

üí° Type Conversion Strategy

Golden Rule: Always be explicit dengan type conversions, terutama untuk user input!

‚ùå Dangerous

# mass is str "2.5"
force = mass * 9.8 # ERROR!
‚úÖ Safe
mass = float(entry.get())
# mass is float 2.5
force = mass * 9.8 # OK!
