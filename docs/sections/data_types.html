Apa itu Tipe Data?

üìö Apa itu Tipe Data?

Tipe Data adalah jenis-jenis nilai yang bisa kamu simpan dalam program. Seperti di kehidupan nyata ada angka, tulisan, benar/salah - di Python juga sama!

Bayangkan seperti kotak penyimpanan: kotak angka (untuk nomor), kotak tulisan (untuk nama), kotak ya/tidak (untuk jawaban). Setiap kotak punya aturan sendiri!

‚úÖ Kenapa Tipe Data Penting?

1. Cegah Kesalahan

Seperti tidak bisa memasukkan air ke tempat pensil - Python juga akan bilang "salah!" kalau kamu salah taruh data.

2. Hemat Memori

Angka kecil butuh ruang kecil, angka besar butuh ruang besar. Python atur otomatis buat kamu!

3. Kode Lebih Jelas

Kalau ditulis umur: int, langsung tau itu angka. Kamu dan teman bisa langsung paham!

4. Lebih Cepat

Pakai tipe data yang tepat = program jalan lebih kencang!

Tipe Dasar (Primitive)
  • ‚Ä¢ int - Bilangan bulat: 42, -17
  • ‚Ä¢ float - Desimal: 3.14, 0.01
  • ‚Ä¢ str - Teks: "Hello", 'Python'
  • ‚Ä¢ bool - Benar/Salah: True, False
Tipe Koleksi (Kumpulan)
  • ‚Ä¢ list - Urut, bisa diubah: [1, 2, 3]
  • ‚Ä¢ tuple - Urut, tetap: (1, 2)
  • ‚Ä¢ dict - Pasangan kunci-nilai: {"x": 5}
  • ‚Ä¢ set - Nilai unik: {1, 2, 3}

üí° Strategi Konversi Tipe Data

Aturan Emas: Selalu perjelas tipe data, terutama saat minta input dari user!

‚ùå Berbahaya (Dangerous)
mass = entry.get()
# mass is str "2.5"
force = mass * 9.8 # ERROR!
‚úÖ Aman (Safe)
mass = float(entry.get())
# mass is float 2.5
force = mass * 9.8 # OK!

Boolean (bool)

Boolean hanya punya 2 nilai: True atau False. Digunakan untuk flags dan conditional logic. 

# app.py - State Flags
self.is_running = False
self.is_paused = False

# app.py - Dictionary with boolean values
self.contact_tracker = {
    "in_contact": False,  # Boolean flag
    "impulse_accumulated": 0.0,
    "force_samples": []
}

# app.py - Penggunaan dalam conditional
if not self.is_running:
    return

if self.is_paused:
    # ...schedule next check

Pattern: Boolean Flags untuk State Management

Boolean flags seperti is_running, is_paused membuat code lebih readable daripada menggunakan integer (0/1). Gunakan prefix is_, has_, can_ untuk boolean variables.
